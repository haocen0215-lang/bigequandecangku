<WORLD_generative_rules_NPC命名指南>
# 名称: NPC命名指南（完整版）
# 用途: 为AI即兴生成各族NPC时提供全面的命名风格参考。AI应根据NPC的种族和出身地，选择对应条目，生成符合文化氛围的名字。本指南提供风格方向和少量示例，而非穷举式名单——鼓励在风格框架内自由创造。


# 第一部分: 人类三国
# 文化基底: 欧洲中世纪晚期至文艺复兴过渡期


人类_韦斯托利亚:
  文化根源: "意大利城邦共和国 + 汉萨同盟商业城市"
  命名结构: "名 + 家族姓（商人和市民阶层普遍有姓）。平民可能只有名 + 职业/出身地绰号。"
  风格描述: "受海洋和商业文化影响，名字简短、明快、带有轻快的元音和清脆辅音。商人家族的姓氏往往来自祖先的职业或贸易品。整体给人精明、务实、开放的印象。"
  典型音节: "克(K/C)、兰(Lan)、维(Vi)、斯(S)、特(T)、尼(Ni)、里(Ri)、诺(No)"
  示例: "男: 克雷格·瓦特梅尔, 维克·赫林, 兰斯·布兰特, 尼科·费舍尔 | 女: 希拉·科尔, 维娜·海格, 艾莉丝·布鲁姆, 贝拉·温德"
  注意: "韦斯托利亚是商会掌权的港口共和国，即使是贵族的名字也偏向务实而非华丽。避免使用过于宫廷化或冗长的名字。"

人类_卡尔德里亚:
  文化根源: "法兰克王国 + 诺曼/安茹王朝的封建贵族传统"
  命名结构: "名 + 领地介词 + 领地名（贵族）。平民用名 + 父名或绰号。教士和学者可能使用拉丁化的名字。"
  风格描述: "大陆最传统的封建王国，名字庄重、古典、音节较多。贵族名字中常带有表示领地的介词（'冯'、'德'、'范'）。名字传承感强——长子往往沿用祖父的名字。整体给人端正、保守、讲究血统的印象。"
  典型音节: "奥(Au)、德(De)、里(Ri)、弗(F/V)、加(Ga)、莱(Lai)、温(Wen)、贝(Be)"
  示例: "男: 奥德里克·冯·莱恩堡, 弗雷德里克·德·蒙塔涅, 加雷斯·温斯洛 | 女: 伊莎贝拉·冯·阿尔滕, 玛蒂尔达·德·博蒙, 埃莉诺·贝尔维"
  注意: "长子与祖父同名是常见传统。农民的名字简短朴实——可能只叫'老汉斯'或'磨坊边的皮特'。"

人类_阿尔登马克:
  文化根源: "条顿骑士团 + 普鲁士军事贵族传统"
  命名结构: "名 + 家族姓。军事贵族的姓氏常带有'冯'。军衔或战绩绰号在非正式场合可替代姓氏。"
  风格描述: "边境军事王国，名字硬朗、短促，多带爆破音和卷舌音，给人坚韧、冷峻、纪律严明的感觉。即使是女性名字也比其他两国更为干练。因与兽族联邦接壤，偶尔会出现东方音节融入的混血名字。"
  典型音节: "格(G)、赫(H)、尔(R)、特(T)、克(K)、沃(Wo)、布(B)、施(Sh)"
  示例: "男: 格哈特·冯·施泰因, 赫尔曼·布兰德, 库尔特·艾森, 沃尔夫·克鲁格 | 女: 格蕾塔·冯·阿德勒, 赫尔加·施瓦茨, 乌尔苏拉·克恩"
  注意: "边境地区可能出现人类-兽族混合命名风格（如人类名+兽族式绰号），这是文化融合的自然结果而非错误。"


# 第二部分: 矮人·杜尔加德联山国
# 文化基底: 日耳曼/北欧工匠文化


矮人:
  文化根源: "古挪威/冰岛命名传统 + 托尔金式矮人命名 + 日耳曼工匠行会"
  命名结构: "名 + 氏族姓。氏族姓通常是复合词，由两个描述性词根组成——可以是祖先的身体特征、传奇事迹、擅长技艺或氏族标志性矿物。"
  风格描述: "沉重、厚实、低沉。名字由两到三个重音节组成，充满爆破音和鼻音。姓氏极有画面感——'铁指'让人想到锻造，'铜须'让人想到形象，'深炉'让人想到家族的古老熔炉。无论实际年龄，矮人的名字听起来都像是饱经风霜的老匠人。"
  典型音节: "索(Tho)、巴(Ba)、隆(Lon)、格(G)、姆(M)、杜(Du)、霍(Ho)、布(Bru)"
  示例: "男: 索林·铁砧, 霍恩·铁指, 格隆·深炉, 巴林·铜须, 杜瓦林·石心 | 女: 迪莎·钢锤, 莫拉·火花, 布琳·金梭, 希尔达·岩纹"
  注意: "矮人的性别从名字上不太容易区分——女矮人的名字同样沉稳有力。氏族姓是身份的核心，被剥夺氏族姓是最严重的惩罚。"


# 第三部分: 精灵·翡叶永森 & 暗精灵·多卡海姆
# 文化基底: 共享精灵语根源，分化后各自发展


精灵:
  文化根源: "托尔金精灵语命名传统（昆雅/辛达林色彩） + 古典梵语/巴利语的音韵美感"
  命名结构: "个人名（通常由两个词根组合而成，每个词根都有含义）。极少使用姓氏——精灵寿命漫长，同名者极少，且彼此都认识。在与外族交流时可能加上家系名或居住地名以示区分。"
  风格描述: "空灵、悠长、充满元音。名字的每个音节都像树叶在风中翻转——轻柔、优雅、带有自然的韵律感。双元音和流音（l, r）频繁出现。名字通常蕴含自然意象——星光、树冠、溪流、晨露。整体给人超然、古老、不染尘埃的感觉。"
  典型音节: "伊(I/E)、洛(Lo)、西尔(Syl)、安(An)、瑟(The)、拉(La)、里(Ri)、恩(En)"
  示例: "男: 伊洛维希, 芬丹·塞瑞安, 瑟兰迪尔, 安罗西恩 | 女: 莉亚娜·伊瑟尔, 加拉希尔, 艾洛温, 席尔维娅"
  注意: "精灵极少离开领地，在外族城市遇到精灵是稀有事件。精灵名字不应有爆破音过多或短促粗犷的感觉。"

暗精灵:
  文化根源: "精灵语基础上的分化变体 + 北欧暗色森林/洞穴文化意象"
  命名结构: "个人名 + 氏族名。与精灵不同，暗精灵因居住在幽暗森林的分散聚落中，发展出了氏族制度，氏族名是重要的身份标识。"
  风格描述: "保留了精灵语的流畅感，但音调更低沉、更内敛。元音偏向暗色调（o, u, a而非i, e），辅音中多了摩擦音和嘶声（sh, z, s, th）。名字中的自然意象从精灵的星光树冠转向菌菇、苔藓、洞穴回声、月影。务实而非空灵——暗精灵的名字像是脚踩在潮湿腐殖土上的声音。"
  典型音节: "萨(Za/Sa)、影(Sha)、莫(Mo)、泽(Ze)、洛(Lo)、恩(En)、瑟(Se)、拉(Ra)"
  示例: "男: 萨洛斯·暗苔, 泽拉恩·菌根, 莫希尔·深壤 | 女: 影歌·灰苔, 洛萨拉·苔穴, 瑟琳·幽菌"
  注意: "暗精灵比精灵更频繁出现在外族城市（尤其纳维里斯和人类北境），命名时应反映其务实直率的性格——名字不如精灵那般悠远空灵，更像是'好用就行'的实在感。"


# 第四部分: 兽族联邦·绥和诸邦
# 文化基底: 东亚多元文化圈
# 按生态文化圈分组指导


兽族联邦_通用规则:
  文化根源: "东亚命名传统（中日韩越），共享基底但各邦因生态和文化差异而有明显的风格分化。"
  命名结构: "姓 + 名。姓在前，名在后。名通常为一到两个字。姓氏数量有限，同姓者很多，名字才是区分个体的关键。"
  风格描述: "整体偏向婉转、含蓄、富有意象。名字多与自然、季节、植物、颜色、品德相关。发音以开口元音和鼻音为主。兽族的名字很少直接描述其种族特征——不会有狐族叫'银尾'或犬族叫'利牙'，这被视为粗俗（类似人类给孩子起名叫'两条腿'）。"
  注意: "十六邦共享东亚基底，但音韵偏好和意象来源有差异。以下按生态文化圈给出细分指导。当不确定某个亚种属于哪个圈时，使用通用规则即可。"

兽族_北部草原区:
  涵盖亚种: "犬邦·鸣原, 狼邦·嗥原"
  文化根源: "蒙古/突厥草原命名传统 + 日本东北地区的朴实感"
  风格描述: "朴实、硬朗、带有风吹草地的开阔感。名字音节少而有力，不追求雅致而追求实用。意象多来自草原、风、奔跑、忠诚、群体。犬邦偏温暖直白，狼邦偏冷峻内敛。"
  典型字: "原, 风, 忠, 信, 岚, 苍, 朔, 凛, 牧, 野"
  示例: "犬邦: 岚原, 忠风, 信田, 朝牧 | 狼邦: 朔凛, 苍嗥, 夜信, 霜原"

兽族_中部丘陵区:
  涵盖亚种: "狐邦·萩岭, 猫邦·霞丘, 薮猫邦·影丘"
  文化根源: "日本平安/室町时代的雅致命名 + 中国宋代文人气息"
  风格描述: "典雅、灵动、带有一丝狡黠或神秘。名字音韵柔和，意象多来自秋叶、月光、雾霭、丘陵、花卉。狐邦偏灵动机敏，猫邦偏慵懒月夜，薮猫邦偏幽深寂静。是联邦中命名最讲究美感的文化圈。"
  典型字: "萩, 霞, 月, 影, 枫, 露, 铃, 岚, 雾, 夕, 琥, 珀"
  示例: "狐邦: 萩铃, 霞枫, 露叶, 琥珀 | 猫邦: 月影, 夕雾, 朦胧, 霞音 | 薮猫邦: 影泽, 暗月, 幽铃, 深丘"

兽族_中南水乡区:
  涵盖亚种: "兔邦·露泽, 鹿邦·幽泽"
  文化根源: "中国江南水乡命名传统 + 日本水乡/田园风"
  风格描述: "清新、温柔、水润。名字带有湿润的质感，意象多来自莲花、溪水、露珠、竹林、药草。兔邦偏活泼明快，鹿邦偏沉静幽远。是联邦中名字听起来最温柔的文化圈。"
  典型字: "莲, 露, 泽, 清, 竹, 菖, 蒲, 幽, 翠, 萱, 菱, 荷"
  示例: "兔邦: 莲露, 清泽, 菱月, 春菖 | 鹿邦: 幽兰, 药泽, 翠萱, 菖蒲"

兽族_南部畜牧区:
  涵盖亚种: "羊邦·牧坡, 牛邦·原泽, 狮邦·烈原"
  文化根源: "中国西北/西南的朴实命名 + 日本乡土气息"
  风格描述: "厚实、温暖、大气。名字给人脚踏实地的安稳感，意象来自丰饶的牧场、厚实的土地、金色的谷穗、温暖的炉火。羊邦偏温软，牛邦偏稳重，狮邦偏响亮豪迈。"
  典型字: "丰, 穗, 牧, 原, 实, 厚, 暖, 烈, 金, 稳, 壮, 雷"
  示例: "羊邦: 暖穗, 柔牧, 绵原, 和泽 | 牛邦: 丰原, 厚实, 稳田, 壮土 | 狮邦: 烈金, 雷原, 壮火, 豪风"

兽族_东南热带区:
  涵盖亚种: "虎邦·虎啸岭, 鳄邦·沼泽三角洲, 蜥蜴邦·石鳞地"
  文化根源: "东南亚命名传统（泰/缅/越） + 中国南方少数民族命名"
  风格描述: "热烈、浓郁、带有丛林和水泽的气息。名字音节可以较多，发音中带有更多的开口元音和鼻化音。意象来自烈日、暴雨、巨树、沼泽、岩石。虎邦偏威猛，鳄邦偏深沉，蜥蜴邦偏隐匿。"
  典型字: "烈, 岩, 沼, 蔓, 焰, 潜, 石, 翠, 炎, 深, 古, 蛮"
  示例: "虎邦: 炎岩, 烈啸, 焰林, 猛原 | 鳄邦: 深沼, 潜渊, 古泽, 沉水 | 蜥蜴邦: 石隐, 岩纹, 伏沙, 暗鳞"

兽族_特殊文化区:
  涵盖亚种: "鸟邦·翠峰, 熊猫邦·翠屏, 狸猫邦·迷原"
  文化根源: "鸟邦: 中国道家/仙人文化的高远感。熊猫邦: 中国四川盆地的悠闲文化。狸猫邦: 日本妖怪传说中的化狸文化。"
  风格描述: "三个亚种风格差异明显。鸟邦名字轻盈高远，带有风和天空的意象；熊猫邦名字圆润温和，充满竹林和闲适的气息；狸猫邦名字带有一种质朴中藏着灵巧的感觉，不起眼但回味悠长。"
  典型字: "鸟邦: 翔, 云, 峰, 风, 羽, 翠, 霄 | 熊猫邦: 竹, 圆, 安, 悠, 翠, 闲, 墨 | 狸猫邦: 迷, 化, 朴, 路, 野, 平, 吉"
  示例: "鸟邦: 翔云, 翠峰, 霄风, 高羽 | 熊猫邦: 竹安, 圆墨, 悠闲, 翠平 | 狸猫邦: 野吉, 路平, 朴化, 迷原"


# 第五部分: 龙裔·阿什卡纳尔
# 文化基底: 中东/北非


龙裔:
  文化根源: "阿拉伯/波斯命名传统 + 古埃及贵族命名"
  命名结构: "个人名 + 血脉家名。血脉浓度高的贵族家族有绵延数百年的家名，血脉浓度低的平民可能只有个人名 + 出生地名。"
  风格描述: "华丽、繁复、充满异域风情。名字中多个音节层叠，带有弹舌音(r)、喉音(kh, gh)和长元音，听起来像沙漠中的热风掠过金色宫殿。男性名字通常刚猛华美，女性名字带有珠宝和花卉的璀璨感。整体给人高贵、炽热、不可冒犯的印象。"
  典型音节: "阿(A)、卡(Kha)、纳(Na)、扎(Za)、尔(R)、拉(Ra)、米(Mi)、萨(Sa)、丁(Din)"
  示例: "男: 阿什卡·拉扎尔, 扎卡里·哈迪尔, 萨拉赫丁·卡西姆 | 女: 娜芙蒂蒂·阿米拉, 阿米娜·萨菲亚, 莎赫拉·法蒂玛"
  注意: "血脉浓度是龙裔社会最核心的等级标志，高血脉贵族的名字通常更长、更华丽。低血脉平民的名字可以简短朴实。"


# 第六部分: 珑族
# 文化基底: 中华古典


珑族:
  文化根源: "中国古典命名传统（先秦至宋），强调家族传承和美德寄托"
  命名结构: "姓 + 名。姓在前，名在后。名通常为一个字，偶尔两个字。姓氏数量有限（约二三十个常见姓），用于标识家族血脉。"
  风格描述: "极简、含蓄、雅致。名字只有一个字，但这个字被精心选择，承载着父母对孩子的全部期望——可能是美玉（玉、瑜、琳）、品德（安、信、义）、自然（泉、岚、霖）、或家庭愿景（平、兴、和）。珑族名字的美在于以少胜多——一个字就是一幅画。"
  典型姓氏: "林, 苏, 陈, 李, 赵, 周, 吴, 何, 张, 沈"
  典型名字用字: "玉, 安, 平, 明, 心, 瑜, 信, 和, 泉, 霖, 琳, 兴"
  示例: "林安, 苏玉, 陈明, 李心, 赵瑜, 周信, 何泉, 张琳"
  注意: "珑族体型极小、面容永远稚嫩，但名字不应因此显得幼稚——一个叫'林安'的珑族可能已经活了一百二十年。名字的庄重感与外貌的反差正是珑族的魅力之一。"


# 第七部分: 半人马·阿斯特拉原野
# 文化基底: 古希腊牧歌 + 星辰崇拜


半人马:
  文化根源: "古希腊命名（牧歌/田园诗色彩） + 星座/天体命名"
  命名结构: "个人名，无姓氏。半人马是游牧部落制，用'某某部落的某某'来区分同名者，但这不是正式姓氏——更像是方位标识。"
  风格描述: "开阔、辽远、带有星辰和原野的气息。名字多为三到四个音节，元音清亮，辅音轻柔，念出来像是在广阔草原上仰望星空时的一声轻叹。意象来自星辰、奔跑、风、地平线、黎明。不分性别——半人马认为名字属于灵魂而非身体。"
  典型音节: "阿斯(As)、特拉(Tra)、利(Li)、恩(En)、菲(Phi)、洛(Lo)、翠(Cy)、安(An)"
  示例: "阿斯特利恩, 菲洛翠安, 洛利翠斯, 特拉西恩, 安菲洛斯, 恩特拉"
  注意: "半人马名字不区分性别。避免使用过于沉重或粗犷的名字——半人马虽然体型庞大但精神世界轻盈高远。"

拉弥亚:

# 第十三部分: 血族

  文化根源: "沿用转化前所属种族的命名传统"
  命名结构: "与转化前种族完全相同。血族的全部生存策略建立在'伪装成原种族'的基础上——使用不符合原种族的名字等于自杀。"
  风格描述: "无独立风格。一个原本是韦斯托利亚商人的血族，名字仍然是韦斯托利亚式的；一个原本是暗精灵的血族，名字仍然是暗精灵式的。唯一的区别在于——极古老的血族（活了数百年以上）可能使用的是'几百年前那个种族的命名风格'，听起来略显古旧过时。"
  注意: "不要让血族的名字暴露其身份——没有'暗夜'、'血月'、'永生'这种明示性的名字。血族的可怕之处恰恰在于，你永远无法从名字上辨认他们。如果需要暗示一个NPC是血族，可以让其名字风格'稍微老派了一点点'——但也仅此而已。"
