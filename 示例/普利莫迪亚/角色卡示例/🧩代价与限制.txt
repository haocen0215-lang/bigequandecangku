代价与限制: # 使用力量的生理代价、装备消耗和战斗逻辑限制，所有代价非永久性但危险环境中失能等于死亡
  枯竭体系: # 力量使用的生理/精神代价，均为四级递进（轻度→中度→重度→完全枯竭），最严重为昏迷失能但可完全恢复
    体力枯竭: # 物理系主要代价；来源: 肌肉撕裂/乳酸堆积/脱水/关节磨损；四级表现: 乏力→肌肉痉挛抽搐→行动能力丧失→器官负荷过大昏迷；特征: 意识可能清醒但身体不听使唤；恢复: 休息/食物/药剂/物理治疗
    魔力枯竭: # 魔法系主要代价；来源: 汲取操控魔力场的精神神经负荷到极限；四级表现: 头痛→感官紊乱（幻听幻视/方向感丧失）→失去施法能力→精神层面昏厥；特征: 身体可能勉强站立但脑子不听使唤；恢复: 休息/魔力恢复药剂/适度魔力环境校准
    特殊能量透支: # 物理系英雄级以上特有；来源: 特殊能量（狂暴/苦修之力/信仰之力）耗尽后的混合衰竭；同时出现体力枯竭和精神疲惫但不等同魔力枯竭因不涉及外部魔力场；恢复比单纯体力枯竭更慢因身心双重损耗
  装备崩坏: # 装备不是永恒的，越强的力量对装备消耗越大
    常规磨损: # 日常使用和普通战斗的缓慢损耗（刀刃钝化/铠甲凹陷/弓弦松弛）；定期维护和工匠修缮可长期保持；所有等级的日常开销
    战斗折损: # 高强度战斗的加速消耗——附魔符文过载失效/材质在超常力量冲击下碎裂/法器魔力灌注过度烧毁；英雄级以上一场可能报废半套装备；高阶冒险者是装备消耗大户
    环境腐蚀: # 深界高浓度魔力场加速非特殊材质物理老化，普通铁制武器在深层数天内可能锈蚀脆化；深界产素材对深界环境有天然抗性——这是深界素材地表价格高昂的核心原因之一
  动作经济学: # 战斗中的时间/体位/注意力成本，再强的存在在动作切换瞬间也有脆弱性
    动作成本: # 每个动作占用时间和体位——越强招式需越大蓄力/引导/精神集中；高阶法术吟唱手势时间更长；特殊能量状态切换需过渡瞬间；力量等级越高执行越快但成本始终存在
    破绽窗口: # 动作切换期间的防御空档——蓄力中防御空虚/施法中近乎无自保/进入狂暴切换瞬间为绝对空档；以弱胜强理论可能但极端罕见——高阶者破绽窗口小到常人无法感知利用，等级差距越大利用难度指数级上升
    团队分摊: # 团队协作的核心价值——前排承担被攻击成本让后排获施法窗口/斥候承担侦察成本减少破绽暴露/治疗者实时管理全队枯竭进度在阈值前介入；配合默契的白银级五人队特定情境下可能胜过独行黄金级冒险者
