荣誉守则:
  基本准则: 不背叛队友、诚实守信（不虚报任务完成情况）、公正分配（战利品按约定分配）、互助精神（遇险同伴必须救援）。
  违规处罚: 初次违反警告 → 再次违反降级并禁足一月 → 严重违反永久除名 → 叛变行为全大陆通缉（标记为"虚无"）。

等级评定标准: # 等级=综合职业资历，非战力排行
  评定维度:
    任务完成记录: # 权重最高。数量/质量/难度/成功率
    信用评级: # 合同履行率/队友雇主反馈
    专业能力: # 不限战斗。采集/向导/测绘/学术/后勤均认可
    工会贡献: # 带新人/救援/情报/经验分享
    同行推荐: # 秘银+晋升需两名同级或更高推荐
    深层经验: # 秘银+需深界中层以上实际活动经验（非必须独自战斗，但需证明深层生存能力）

  力量与等级关系:
    统计相关非硬绑定: # 高等级中强者比例高（深层报酬高→深层需实力→强者更易积累资历），但个体可偏离
    低阶可脱钩: # 刚入会大师可能只是黑铁；见习力量老牌采集专家可能是白银
    高阶收敛: # 山铜/精金几乎必然大师+力量——非门槛而是自然结果（深层远征资历本身要求实力）

  晋升流程:
    黑铁→白银: # 积分自动晋升
    白银→黄金: # 申请+分部审核
    黄金→秘银: # 审核+一名秘银+推荐+分部会长审批
    秘银→山铜: # 两名山铜+推荐+总部审批
    山铜→精金: # 山铜/精金联名推荐+总部高层评议
    精金→以太: # 理论存在，从未授予

  降级与除名:
    严重失信: # 降一至两级
    违反铁律: # 按荣誉守则处罚


公会传统:
  重要节日:
    冒险者之日（花萌月15日/春分）: 新人入会仪式，老冒险者分享经验，传承冒险精神。
    群星试炼（金叶月15日/秋分）: 全大陆竞技大会，项目包括战斗、解密、生存。冠军可获直升一级的荣誉。
    缅怀之夜（暗夜月21日/冬至）: 纪念牺牲的冒险者，点燃长明灯，遗属获得抚恤金。

经济职能:
  任务佣金: 标准抽成10%，高危任务降至5%。
  材料交易所: 设于纳维里斯总部和各主要城市分部，提供挂牌交易、竞价拍卖、质量认证。
  信用体系: 根据任务完成率、还款记录、公会贡献综合评定信用等级，影响可接取任务等级和贷款条件。

服务体系:
  医疗救援: 各分部配备医疗牧师。
  装备维护: 武器保养、护甲修复、附魔服务、装备鉴定。
  情报整理部: 提供魔物动向、遗迹线索等情报，按重要性分级定价。
  档案管理: 冒险者档案、任务存档、魔物图鉴、地形图，按等级开放查阅权限。

分部体系:
  总部: 纳维里斯，直接管理深界主入口。
  各族分部: 设在各种族主要城市，管理当地裂隙入口。
  裂隙入口岗哨: 设在每个已知裂隙入口旁，最低级别的工会存在——可能只有几名工会人员负责登记、公示规则、防止平民误入。非任务联络点。
