气味标签: # 菜品的气味维度标准词汇表。为每道菜品挂载气味标签，包含正面和负面气味。

  # ——正面·热反应系——
  焦香: # 美拉德反应产生的深色暖香，煎烤肉类表面、烤面包壳
  焦糖甜香: # 糖或蜂蜜受热焦糖化产生的琥珀色黏稠甜香
  烟熏香: # 木炭柴火烟气渗入食材的木质气息
  炭烤香: # 比烟熏更直接的明火灼烧气息，带微苦底味
  锅气: # 铁锅极高温瞬间产生的独特焦香，炒菜专属

  # ——正面·油脂系——
  奶香: # 黄油牛奶奶油的温润乳脂气息
  榛果香: # 黄油褐化后产生的坚果暖香
  荤脂香: # 动物脂肪受热释放的浓郁肉香
  骨髓香: # 骨头长时间熬煮释放的浓厚油脂底香
   乳化奶香: # 蛋黄或黄油经乳化后特有的轻盈丝绒气息，比纯奶香更柔和更有空气感

  # ——正面·植物系——
  草本香: # 迷迭香百里香月桂叶薄荷等的清凉木质气息
  辛香: # 黑胡椒辣椒肉桂丁香姜等香料的温热穿刺气息
  果香: # 水果的清甜酯类气息
  花香: # 藏红花洋甘菊接骨木花等的淡雅芬芳
  蔬菜甜香: # 洋葱焦糖化或根茎烤软后的温和植物甘甜
  青草气: # 新鲜叶菜香草刚切开时的生鲜绿意
  坚果香: # 杏仁榛子核桃烘烤后释放的温暖油质香
  麦芽甜香: # 谷物发芽或轻烘的基底暖香
  茶香: # 茶叶的清幽气息，介于草本清新与花香淡雅之间

  # ——正面·发酵深香系——
  酒香: # 烹饪用酒酒精蒸发后留下的醇厚发酵气息
  酱香: # 鱼露酱油豆酱等发酵调味料的浓郁咸鲜嗅觉
  菌菇香: # 菌菇受热释放的潮湿泥土般幽深气息
  海潮气: # 海产海盐海藻携带的碘味咸雾气息
  发酵酸香: # 泡菜酸面团酸奶等良性发酵的清洌微酸气息
  陈醋香: # 好醋特有的深沉圆润酸香
  豆酱香: # 大豆或豆类经长期发酵产生的浓郁深沉气息，咸鲜底味厚重，带微甜回味
  鱼酱香: # 鱼虾经盐渍发酵产生的极致鲜香，海潮气与发酵深香交织，浓度极高
  果酱香: # 水果经长时间熬煮浓缩后的甜蜜浓稠气息，果糖焦化的温暖底味


  # ——负面·烹饪失败系——
  焦糊味: # 食材烧焦碳化后的刺鼻苦臭
  油烟味: # 油脂过热超过烟点后分解产生的呛人灰烟
  生腥味: # 肉类血水未处理或鱼类去腥失败的铁锈腥气
  膻味: # 羊肉鹿肉等野味特有的动物体脂浓烈气息
  酸腐味: # 食材不新鲜或发酵过度的腐败酸气
  霉味: # 菌菇或食材储存不当的阴湿腐朽气息
  糊锅味: # 锅底食材粘连烧焦混入菜中的焦苦底味
  血腥气: # 肉类未充分加热时散发的生血铁锈气息
  蛋腥气: # 蛋类加热不当或过量使用时的硫化物气味
  油哈味: # 油脂氧化变质产生的陈旧刺鼻酸败气息

  # ——负面·搭配冲突系——
  甜腥气: # 甜味食材与未去腥水产混合产生的恶心混合气息
  腻浊气: # 油脂过多缺乏酸辛调和时的沉闷厚重气息
  杂乱气: # 过多强烈气味互相冲撞无法融合的混沌感
  甜膻气: # 甜味与羊肉膻味叠加产生的奇异不适气息
  酸腥气: # 醋酸与鱼腥交织产生的刺激性混合气息
