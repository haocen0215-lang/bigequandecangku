# 名称: 深界层域档案生成器
# 用途: 一个复合型生成规则，用于系统性地创造深界中每一个独立世界层（层域）的完整设定。它结合了定义所有层域共享底层规则的"深界共性法则"、用于规范每层设定的"世界层档案模板"、以及用于激发创意的"层域灵感材料库"。

# ═══════════════════════════════════════
# 第一部分: 深界共性法则
# ═══════════════════════════════════════

深界共性法则:

  基本性质:
    定义: 深界是地表之下的、由多个完整平行世界构成的广袤存在。每个世界层（称为"层域"）拥有自己的天空、光源、日夜周期、地貌、气候和完整生态系统。层域不是"洞穴"或"地下城"，而是踏入后抬头可见天空、脚下可踩大地的独立世界。
    数量: 理论上无限。无人知晓最深处。当前地表文明已确认存在至少二十个层域。
    美感定位: 深界整体比地表更美、更壮丽。这种超越性的美是驱动冒险者深入探索的核心动力之一——不仅为了财富，也为了亲眼见到地表绝不可能存在的景色。

  入口与空间映射:
    主入口: 位于纳维里斯城正下方，是最大、最稳定的通道。
    裂隙入口: 以纳维里斯为中心向外扩散分布于各种族领地，由当地冒险者工会岗哨管理。数量众多，规模不等——大城市附近的入口宽阔稳定，偏远村庄附近的可能仅容单人通过。
    空间映射法则: 每个层域的空间范围极其广袤，覆盖面积与地表大陆的入口分布范围大致匹配。各地表入口在层域中出现的位置与地表地理方位大致对应——韦斯托利亚的入口通向层域的西部区域，兽族联邦的入口通向层域的东部区域，纳维里斯主入口大致位于层域的中央。不同入口之间在层域内的距离极远，各自形成独立的活动区域。

  通道物理形态:
    基本性质: 连接地表与层域、以及连接相邻层域之间的通道，是实实在在的、有物理长度的走廊/隧道/斜坡。通道有足够的宽度和高度，可以安装基础设施——铁轨、矿车、蒸汽提升设备、照明、通风等。
    通行时间参考:
      地表→第1层域: 15分钟-30分钟（主入口快，偏远裂隙入口慢）
      第1→第2层域: 1-2小时
      第2→第3层域: 1-2小时
      第3→第4层域: 3-5小时
      更深层域间: 随深度递增，通道更长更复杂
    规模差异: 纳维里斯主入口通道最为宽阔，已被改造为大型通道——支架支撑穹顶、铺设轨道。矮人铁砧山脉的入口通道是全大陆工业化程度最高的——矮人为矿工设计了蒸汽卷扬机和提升系统，专门用于搬运重型矿石。其他种族的入口通道工业化程度不一，偏远裂隙入口可能只是天然的狭窄岩缝。
     运输能力分级:
    说明: 通道的天然物理形态（宽度、高度、地面平整度、坡度、弯道半径）决定了其运输能力上限。大部分通道是天然形成的，尺寸参差不齐。工业化改造是后天的、局部的、昂贵的——仅有少数高投资通道被人工扩建。
    运输等级:
      铁轨级: 通道足够宽阔平整，可铺设铁轨和运行矿车/蒸汽提升设备。仅存在于纳维里斯主入口和矮人铁砧山脉方向的部分通道——均经过数十年人工扩建。
      驮畜级: 通道宽度和地面条件允许驮马或驮驴通行。天然具备此条件的通道不多，部分经简易改造（铺设木板、凿平突出岩石）达到此标准。
      手推车级: 通道允许人力手推车通过。大部分主要城市附近的入口通道天然达到或接近此标准。
      徒步级: 仅允许徒步通行。人员可以背负装备和物资，但无法使用任何轮式或畜力运输工具。大部分层域间通道的天然状态属于此等级。
      攀行级: 通道陡峭、狭窄或地面极不规则（如树根表面、藤蔓网络、岩缝），需要攀爬或特殊技巧才能通过。部分深层通道和偏远裂隙入口属于此等级。
    已知通道运输等级速查:
      地表→翠风原野（第1层）: 纳维里斯主入口=铁轨级 | 矮人方向=铁轨级 | 主要城市入口=手推车级 | 偏远裂隙=徒步级
      翠风原野→深岩矿域（第2层）: 大部分=手推车级（天然岩洞斜坡）| 矮人方向=铁轨级（人工扩建）| 部分=驮畜级
      深岩矿域→千柱深林（第3层）: 大部分=攀行级（树根隧道，地面不规则）| 高流量路线=徒步级（部分铺设木板步道）
      千柱深林→万蔓雨林（第4层）: 大部分=徒步级至攀行级（藤蔓隧道，需定期维护保持通行）
      万蔓雨林→焰潮群岛（第5层）: 攀行级（火山岩裂缝+空间方向反转，仅限有经验的徒步者）
      焰潮群岛→澄镜牧野（第6层）: 徒步级（火山岩洞穴至草皮通道，宽度仅容两到三人并行）

    经济影响: 通道运输等级的逐层衰减是深层资源价格高昂的核心物理原因之一。越深层的资源，运输成本在最终售价中的占比越高。这也解释了为什么矮人对蒸汽卷扬机和通道基建的投资具有战略意义——他们是唯一系统性提升通道运输等级的种族。


  层域间通道:
    多入口法则: 每个层域中通往下一层域的入口不止一个，同样分布在层域各处。所有这些入口都通向同一个下层世界的不同位置。
    通道预示法则: 从第N层域通往第N+1层域的通道，其内部环境（气味、温度、湿度、光线、壁面材质、声音）会在通道的物理长度中从当前层域的特征逐渐过渡为下一层域的特征。通道是两个世界之间的"渐变带"——行走数百米到数公里的过程中，冒险者能感受到下一个世界的"味道"逐渐替代当前世界。
    通道外观法则: 通往下一层域的通道入口在外观上与下一层域的环境特征相关联——如果下一层是水域世界，通道入口可能是一面水幕或潮湿的岩缝；如果下一层是巨型植物世界，通道入口可能是参天古树的树洞。

  线性法则:
    基本原则: 必须穿过第N个层域才能到达第N+1个层域。返回同理——从第N+1层域必须回到第N层域才能继续向地表返回。不可跨层直接返回。
    陷阱通道: 存在极少数隐蔽的陷阱性通道，可能导致冒险者意外跳过一个或多个层域，直接坠入更深的世界。这类通道极度危险——跳过的层域仍需逐层返回，意味着冒险者可能陷入超出自身等级的世界。

  返回法则:
    规则: 进入深界24小时内可原路返回地表。超过24小时后，无论走哪条路，所有通往地表的通道只通向纳维里斯主入口。
    适用范围: 此规则对通过通道的一切事物生效，不区分智慧生物和无生命物体。人和货物遵循相同法则。
    性质: 此规则无物理解释，是世界的固有法则。

  浅层通勤与深层远征:
    说明: 返回法则与通道通行时间共同创造了深界经济的核心分野。
    浅层通勤模式（第1-3层域）:
      描述: 各种族领地的矿工、采集者、猎人通过本地入口进入深界，工作十数小时后在24小时内原路返回。如同日常上下班。
      资源流通: 浅层资源从各地入口分散流出，在各自的地表市场流通。不依赖纳维里斯。
      经济影响: 浅层资源价格相对低廉，供应相对充足。各种族通过本地入口独立经营，自给自足。
      基础设施: 各种族根据需求在本地入口通道安装设备。矮人入口工业化程度最高（蒸汽卷扬机），其他种族程度不一。
    深层远征模式（第4层域及更深）:
      描述: 到达第4层域及更深需要穿越前面数个层域，单程时间远超24小时。远征队在深界停留数天到数周。
      资源流通: 超过24小时后所有返回通道只通向纳维里斯——因此深层稀有资源全部且只能从纳维里斯流出。
      经济影响: 纳维里斯是深层稀有资源的唯一出口城市。这是纳维里斯经济繁荣和地缘政治紧张的根源。远征队通常从纳维里斯出发（否则返回后还需从纳维里斯长途跋涉回家）。
      基础设施: 纳维里斯主入口通道工业化程度最高，配套远征补给、物资提升、矿车轨道等完整体系。

  深度递进:
    壮丽递进: 越深的层域越美、越壮丽。浅层世界像是"比地表更好的地表"，深层世界则展现地表完全无法想象的景观。
    危险递进: 越深的层域环境威胁和生物威胁越大，对应更高的冒险者等级要求。
    资源递进: 越深的层域资源越稀有、越有价值、在地表售价越高。
    魔力递进: 越深的层域魔力场浓度越高。

  资源反直觉递进:
    定义: 深界资源（食材、矿物、草药等所有类别）的外观/名字/第一印象与实际性质之间的偏差程度，随层域深度递增。
    反直觉类型:
      符合直觉: 外观和实际性质一致，与地表同类资源的认知逻辑相同。
      部分反直觉: 某一个感官维度与直觉相反（如看起来像肉但闻起来是花香，口感仍是肉）。
      完全反直觉: 多个维度反转（如外观是暗红尖椒、触感温热，实际咬下去是甜味多汁水果）。
      不可预测型: 既不正也不反，是地表完全没有对应物的全新感官体验，无法用地表经验判断。

    深度比例参考:
      第1-3层域: 符合直觉70% / 部分反直觉20% / 完全反直觉10% / 不可预测0%
      第4-6层域: 符合直觉40% / 部分反直觉30% / 完全反直觉20% / 不可预测10%
      第7-13层域: 符合直觉20% / 部分反直觉30% / 完全反直觉30% / 不可预测20%
      第14-18层域: 符合直觉10% / 部分反直觉20% / 完全反直觉35% / 不可预测35%
      第19层域: 符合直觉5% / 部分反直觉10% / 完全反直觉35% / 不可预测50%
      第20层域：无存活，完全未知
备注: 以上比例为参考基准。特殊用途层域（如以安全/休憩/居住为主要功能的层域）可适度偏离参考比例，在档案的反直觉程度说明中注明偏离原因即可。

层域命名灵感池:
  自然意象系: "以该层域最显著的自然特征命名，如特定颜色的天空、标志性的地貌、永恒的季节。例如: 翠穹原野、琥珀黄昏、霜花旷谷。"
  感官体验系: "以踏入该层域时最强烈的感官印象命名。例如: 静寂之海、回响高原、薄荷晨雾。"
  冒险者俚语系: "民间和冒险者群体自发产生的、更口语化的别称。例如: 新手草坪、蘑菇地狱、宝石大教堂、白鲸天。"
  诗意/文学系: "带有文学色彩和情感的命名，往往来自学者或吟游诗人。例如: 遗落的黎明、星河摇篮、沉睡者的花园。"
  工业/功能系: "以该层域的主要经济用途或人工改造特征命名。例如: 铁脉走廊、矿工走道、蒸汽深坑。"

天穹形态池:
  单日型: "一颗类似太阳的光源，但大小、颜色、轨迹可能与地表不同——更大、更柔和、偏橙或偏蓝。最接近地表体验的天穹。"
  双日/多光源型: "两个或以上光源同时存在于天空中，可能颜色不同、亮度不同、运行轨迹交叉。投射出多重影子。"
  无日恒光型: "天空本身发出均匀的漫射光，没有可辨认的光源。像永远的阴天但明亮温暖，不投射明确的影子。"
  永恒时刻型: "天空凝固在某个永不变化的时段——永恒的黎明、永恒的正午、永恒的黄昏、永恒的满月夜。没有日夜交替。"
  星海型: "天空中布满极其密集且明亮的星辰，地面亮度接近满月之夜但色彩更丰富。可能有可见的星云和光带横跨天际。"
  天幕生物型: "天空中有巨大的发光生物（鲸、水母、鸟群等）缓慢游弋或迁徙，它们的生物光构成这个世界的主要光源之一。"
  极光/流光型: "天空中持续展示极光般的色带流动或光柱旋转，光的颜色和强度有缓慢的节律变化。"
  穹顶/封闭型: "没有传统意义上的天空。头顶是岩石穹顶、巨型树冠网络、或其他遮蔽结构。光源来自人工照明、发光矿脉、荧光生物等。"

地貌池:
  温带田园型: "最接近地表的地貌——草原、丘陵、森林、溪流——但一切都更茂盛、更鲜艳、更生机勃勃。适合浅层世界。"
  巨型植物型: "数百米高的巨大树木、蘑菇、花朵构成主要地形。地面是它们的根系和落叶层。仰头看到的是枝叶间透下的光柱。"
  浮岛/天空之城型: "大地破碎为大小不一的岛屿漂浮在空中，之间由天然桥梁、藤蔓、瀑布或风流连接。从边缘望下去是云海或更多浮岛。"
  水晶/矿化平原型: "地面由半透明或发光的矿物、水晶构成。光线在矿化地表中折射，地面本身像一面棱镜。"
  巨型水域型: "大部分面积被水覆盖。陆地以零散的岛屿、浅滩、珊瑚礁式的凸起形式存在。水面可能有独特颜色。"
  花海/永恒花园型: "一望无际的花田或植被覆盖，品种和色彩超出地表想象。可能只有一种花也可能万花齐放。空气中弥漫花粉和香气。"
  峡谷/悬崖/层叠型: "地形以深谷、高崖、层叠台地为主。垂直空间感极强。从崖顶俯瞰可以看到数层台地如阶梯般向下延伸。"
  熔岩/地热型: "地热活跃的世界——间歇泉、温泉、熔岩河流。但不一定阴暗可怖，可能是温暖而瑰丽的火山景观。"
  洞窟/隧道网络型: "巨大的天然或人工洞穴体系构成主要空间。没有传统天空，光源来自矿脉辉光、人工照明或发光生物。可能包含极高极宽的主厅和错综的支巷。"

水体池:
  清澈河流/溪流型: "类似地表河流但水质极其清澈，可能折射出不寻常的颜色或隐约发光。"
  大型湖泊/内海型: "平静的大型水面，完美倒映天穹。水下可能有自己的光源系统，使湖面呈现双层景色。"
  瀑布群型: "多个大型或巨型瀑布。水量充沛，制造大量水雾和彩虹。"
  温泉/矿泉型: "温暖的、富含特殊矿物的水体。可能有疗愈效果。颜色可能是乳白、翠绿、淡蓝等。"
  特殊液态型: "不完全是'水'——可能是更黏稠、更轻盈、带有微光或特殊气味的液体，但承担着水在生态中的角色。"
  雨/降水型: "该层域有频繁的降水现象，雨水可能有特殊性质——发光、温热、带有花香、落地后变成特殊植物。"
  地下水/渗水型: "岩壁渗水、地下暗流、矿泉眼。没有大规模地表水体，水以渗透和涓流形式存在。"
  无水型: "该层域没有自然水体或降水。生态以其他方式获取水分（如空气湿度极高、植物储水）。"

通道风格池:
  古门/遗迹型: "巨大的石门或金属门扉，刻满苔藓覆盖的未知符文或浮雕。通道是有物理长度的走廊，行走过程中环境逐渐变化。"
  光之回廊型: "周围的空气开始发光，脚下的地面变得半透明。岩壁退去，走在柔和光芒构成的长廊中。行走数百米后抵达新世界。"
  巨树/根脉型: "巨树的中空树干或根脉形成的隧道。行走过程中木质纹理渐渐变化，空气味道逐渐替换。通道可能长达数公里。"
  水幕/镜面型: "通道入口是瀑布或静水面，穿过后进入湿润的长廊。行走过程中湿度和温度逐渐过渡到下一世界的特征。"
  裂隙/深渊型: "岩壁或地面的裂缝向下延伸为倾斜的通道。有的宽阔可铺设轨道，有的仅容侧身通过。"
  迷雾/渐变型: "浓雾或孢子云包围的长廊。行走过程中周围景色一点点变化——壁面材质、空气温度、光线色调逐渐变得陌生。"
  螺旋阶梯型: "向下的旋转阶梯或斜坡。石壁上的图案和材质随深度变化。每转一段，空气的味道和温度都在改变。宽阔的螺旋通道可铺设轨道。"
  工业化隧道型: "已被人工改造的通道——钢铁支架支撑、铺设铁轨、安装照明和通风设备。蒸汽设备的喘息声和矿车的咔哒声回荡其中。适合描述主入口或高流量通道。"

反直觉维度池:
  - "味觉反转: 外观暗示的味道与实际味道相反（如看起来辣实际甜，看起来苦实际鲜美）"
  - "质地反转: 外观暗示的硬度与实际相反（如看起来梆硬的金属实际柔软如面团，却又打不碎）"
  - "气味反转: 嗅觉印象与实际风味或用途相反（如闻起来恶臭但入口极鲜美，或闻起来芬芳但味道苦涩）"
  - "温度反转: 触感温度与实际效果相反（如摸起来滚烫但吃下去冰凉沁心，或看起来结霜但入口温热暖身）"
  - "时序变化: 入口第一感受在咀嚼或使用过程中完全转变为另一种（如先甜后辣，或先柔软后逐渐变脆）"
  - "用途反转: 外观暗示的用途与实际用途完全不同（如看起来像草药实际是极好的食材，或看起来像宝石实际是调味料）"
  - "外观反转: 所属的生物或物质类别与外观暗示不同（如看起来像矿石实际是植物果实，或看起来像花实际是动物）"
  - "效果反转: 外观暗示的效果与实际效果相反（如画着骷髅图案的草药是强力治疗药，或看起来晶莹剔透的美丽果实有轻微毒性）"
