矮人·日常参考（普里莫迪亚·铁砧山脉·杜尔加德联山国）:

  # 一、基础生理特征

  体型数据:
    女性: 135-145cm / 60-75kg
    男性: 140-155cm / 75-95kg
    体型特点: 骨架极其宽厚·肌肉密度与骨密度极高·体脂极低 / 绝对力量与抗击打能力惊人 / 底盘极稳·重心极低 / 手掌宽大粗糙·指力恐怖
    寿命: 200-300年
    种族定位: 大陆最矮壮精悍的种族。与大体型种族交往时虽需仰视，但其如岩石般的厚重感和恐怖的握力，绝不会让任何人产生“弱小”的错觉

  身材轮廓: 敦实厚重 / 宽肩·厚背·水桶腰（全是核心肌肉）·双腿粗壮如柱 / 胸部极其丰满且坚挺·视觉重心在充满力量感的双臂与厚实的躯干 / 常年挥舞铁锤与开采矿石塑造的极致力量型

  面部要点: 方脸或国字脸·下颌骨极其宽大 / 瞳孔深邃·多为矿石色系（铁灰/琥珀/深褐/暗金）/ 嘴唇厚实紧抿 / 鼻梁宽阔挺拔 / 整体显沉稳·饱经风霜·无论实际年龄多大，总带着一种历经世事的沧桑感

  骨骼与肌肉密度（替代角/耳/尾条目）:
    形态: 骨骼粗壮程度远超其他种族，肌肉纤维极其紧密 / 即使是女性，手臂和大腿的围度也极其惊人
    活动性: 爆发力略逊，但绝对力量和持久耐力无与伦比 / 能轻易举起自身体重数倍的重物
    文化意义: 力量与技艺的基石。矮人之间表达友好的方式通常是极其用力的拍肩或握手（常让外族骨折）。伴侣间喜欢互相揉捏对方坚硬的肌肉，视为极度安心的依靠

  皮肤与抗性（替代角/耳/尾条目）:
    形态: 皮肤极其厚实粗糙，布满老茧和细小的烫伤/划伤疤痕 / 颜色多为古铜、深灰或岩石般的暗褐
    活动性: 对高温、严寒和物理打击具有极其恐怖的抗性 / 能在极高温度的锻造炉旁赤膊工作数小时而不流汗
    文化意义: 伤疤是矮人的勋章。伴侣间在私密时刻互相亲吻对方身上的旧伤疤，是极度深情与敬重的最高表现

  肤色分布:
    色谱: 灰白→古铜→深褐·整体偏暗偏沉
    质感: 粗糙坚韧·如风化的岩石·触感温热且极具摩擦力
    特殊: 掌心、指节和手肘有极其厚重的老茧——常年握持铁锤和机械工具的证明

  毛发特征:
    头发色谱: 铁灰/纯黑/暗红/深棕·极少浅色
    头发质地: 极其粗硬浓密·多为直发或微卷 / 发量惊人·极难打理
    长度文化: 女性多将长发编成极其复杂的粗大麻花辫，并用金属环扣固定·男性多留长发和极其浓密的胡须（胡须是男性矮人的尊严）/ 极度注重毛发的防火处理
    体毛: 极重·四肢和胸腹部有明显的浓密体毛·阴部毛发极其粗硬浓密且覆盖范围极广
    胡须: 男性胡须极其浓密且长及胸腹，常编成辫子并装饰金属扣。女性矮人无胡须，但鬓角和汗毛较重

  体味:
    浓度: 浓郁且厚重
    基调: 燃烧的煤炭+机油+炽热的金属+淡淡的麦酒香
    唤起变化: 极其明显·体温飙升（如烧红的铁块）·散发出极其浓烈的荷尔蒙与金属交织的炽热气息


  # 二、感官能力基准

  感官能力:
    视觉: 中等 / 暗视能力极佳（常年在地下矿洞工作），但对强光和远距离物体不敏感
    听觉: 敏锐 / 能通过敲击声精准判断金属的纯度、机械的故障和岩层的结构
    嗅觉: 中等 / 对瓦斯、毒气和矿物气味极其敏感
    味觉: 极度偏好重口味、高热量食物和极其浓烈的烈酒（尤其是矮人黑啤酒）
    触觉: 极端分化 / 对轻柔的抚摸几乎无感（皮糙肉厚），但对重压、高温和金属的质感极其敏感且享受
    总评: 听觉（材质感知）与触觉（重压/高温）双主导。极度依赖踏实的物理接触和机械的轰鸣声来获取安全感


  # 三、年龄阶段外观

  核心特征: 极其长寿·青春期漫长·衰老体现在胡须/头发变白和动作变得更加沉稳 / 150岁依然能抡起百斤重的铁锤

  幼年期（0-30岁）:
    像一块结实的石头·力气极大·喜欢在矿渣堆里打滚
    开始接触基础的锻造工具，对金属有本能的狂热

  青少年期（30-60岁）:
    身形迅速横向发展，肌肉变得极其夸张 / 女性初潮约在40岁左右
    开始在氏族工坊中担任学徒，学习复杂的机械与符文知识

  青年期（60-120岁）:
    力量与技艺的绝对巅峰
    社会节点: 60岁成年。杜尔加德联山国的核心劳动力，掌控着大陆最顶尖的锻造与蒸汽技术

  壮年期（120-200岁）:
    气质如山岳般不可撼动·技艺达到化境
    社会节点: 氏族议会的议员、锻造大师或蒸汽工程师。多在此时期拥有极其稳定的伴侣

  老年期（200-300岁）:
    头发和胡须变得雪白·皮肤如干裂的树皮
    社会节点: 坐在地热温室旁抽烟斗的智者·掌握着失传的古代锻造配方


  # 四、跨种族关系态度

  矮人看大体型种族: 惺惺相惜 / 觉得牛族或约顿是极佳的酒友和搬运工·喜欢和他们比拼腕力和酒量
  矮人看精灵: 极度嫌弃其矫情和脆弱 / "连把铁锤都拿不动，光长得好看有什么用？"·但在魔法铭刻上偶尔会进行不情愿的学术交流
  矮人看人类: 复杂的商业伙伴 / 觉得人类贪婪且寿命太短，但极度依赖人类庞大的市场来倾销蒸汽机械和武器
  矮人看兽族: 觉得她们过于吵闹和轻浮 / 但对部分力量型兽族（如虎族、狼族）的战斗力表示认可

  跨种族生育:
    矮人×任何种族: 极其罕见。矮人极度看重氏族血脉和锻造传承，与外族通婚被视为"背叛铁砧"。
    若发生: 后代多表现为极其矮壮的人类或带有微弱兽族特征的矮人，在联山国会被边缘化，通常只能前往人类城市谋生。


  # 五、着装速览

  着装:
    核心: 极其耐磨·防火·多口袋 / 厚重的皮革、帆布和金属锁子甲为主 / 色调沉闷——铁灰、煤黑、深褐、暗铜
    日常: 极其宽大的防火皮围裙·带有无数口袋的工装裤·镶嵌钢板的厚底皮靴 / 极度追求实用，身上挂满各种扳手、量尺和齿轮
    正装: 带有氏族徽记的重型板甲或华丽的丝绒长袍（仅限极少数场合）·佩戴极其粗大、镶嵌着极品宝石的纯金/秘银项链
    外出: 随身携带巨大的铁锤或火枪 / 喜欢戴带有护目镜的厚重皮帽
    内衣: 极其粗糙的厚棉布裹胸和短裤 / 女性因为胸部极其巨大且沉重，通常用带有金属搭扣的宽大皮带紧紧束缚以防劳作时晃动
