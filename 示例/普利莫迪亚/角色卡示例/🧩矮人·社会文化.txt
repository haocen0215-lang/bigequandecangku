矮人·社会文化（普里莫迪亚·铁砧山脉·杜尔加德联山国）:

  # 一、图尔加信仰对性文化的影响

  核心: 矮人奉图尔加（炉母）·最侧重"转化、锻造与坚韧" / 性是"两块金属在高温下熔接、锻打成更强合金的神圣过程"
  结果:
    - 极度务实与粗犷: 对性毫不扭捏，将其视为与吃饭、打铁一样消耗体力并带来愉悦的日常活动 / "炉火旺了就要打铁，身体热了就要发泄"
    - 力量与耐力的底层逻辑: 极度推崇强壮的肉体和恐怖的持久力。女性极其丰满厚实的胸臀和粗壮的大腿被视为炉母最完美的杰作，是全族审美的最高标准
    - 同性关系: 接受且包容·只要能一起喝酒、一起打铁、在床上能互相满足，性别并不重要
    - 强迫行为: 极度罕见。矮人性格极其固执且武德充沛，任何强迫行为都会演变成一场砸烂半个酒馆的致命斗殴


  # 二、婚恋与伴侣制度

  成年标准: 60岁
  伴侣制度:
    形制: "熔接礼"·双方在氏族长辈的见证下，共同挥舞铁锤，将两块代表各自氏族的金属锻打成一件完美的器物（通常是戒指或匕首）
    一夫一妻: >98%·绝对主流·矮人极其固执且长情，一旦认准了伴侣，就像焊死的齿轮一样绝不分离
    多偶: 极度排斥·"一台机器装不下三个主齿轮，会卡死的"
    独身: 常见·许多矮人将一生奉献给锻造炉和蒸汽机，认为机械比伴侣更忠诚
    离异: 几乎不可能。除非伴侣背叛了氏族或在锻造上弄虚作假，否则即使天天吵架打架，也绝不离婚

  感情发展:
    节奏: 极其缓慢·充满火药味与酒精味 / 经常是在酒馆里拼酒或在工坊里争论图纸时擦出火花
    青梅竹马: 绝对主流·"从小一起挖矿"是最坚实的感情基础
    表达方式: 极其硬核。送给对方一把亲手打造的、削铁如泥的极品战斧·在对方喝醉时默默把对方扛回房间


  # 三、性产业

  联山国内部: 存在极其粗犷的"红灯酒馆"·通常与烈酒和赌博绑定。提供服务的矮人多为单身且精力过剩者，不涉及道德污点，更像是一场明码标价的体力发泄与肉体搏击
  在外族城市: 极少参与。矮人对其他种族（尤其是人类和精灵）纤细脆弱的身体毫无兴趣，生怕一不小心把对方弄折了


  # 四、亲密社交规范

  公共亲密行为:
    矮人之间: 极其粗暴的肢体接触 / 见面大力头槌（用额头互撞，常发出巨响）/ 伴侣间在酒馆里喝高了，一方把另一方按在桌子上狂吻是日常风景
    拥抱规范: 矮人的拥抱是"窒息式的骨骼挤压"，常伴随着极其用力的拍背，需要伴侣有极强的抗击打能力

  裸体观念:
    极度开放·毫无羞耻感 / 在地热温室或高温锻造炉旁，男女全裸上身、大汗淋漓地工作是常态 / 在地下温泉里集体裸浴是氏族日常
    例外: 极度鄙视那些为了展示身材而穿得花里胡哨、毫无防护作用的衣服

  对情趣道具的态度:
    极度狂热且充满硬核工业风 / 将其视为"检验机械设计与材料强度的绝佳实验"
    典型反应: 嫌弃市面上的道具太脆弱，会亲自画图纸，用黄铜、秘银和蒸汽小型马达打造出极其恐怖、动力强劲的重型情趣机械
    偏好: 喜欢使用带有强烈震动、高温加热甚至齿轮旋转功能的重型金属道具，极度排斥脆弱的硅胶或木头


  # 五、生育观

  社会态度: 极度渴望繁衍·认为强壮的幼崽是氏族未来的铁锤 / 幼崽由整个氏族共同抚养，矮人母亲的奶水极其充沛且营养极高
  熔炉期（极度亢奋机制）: 矮人特有的生理节律。在完成了一件极其完美的锻造作品，或饮用了大量极品烈酒后，会进入为期数天的"熔炉期"。体温飙升，精力极其恐怖，对伴侣的渴望达到狂暴的程度 / 此时受孕率最高
  未婚怀孕: 完全不当回事·"有了就生下来，氏族多把铁锤"·随后补办熔接礼即可
  跨种族后代: 极其护短。如果孩子体型娇小，会被矮人父母疯狂投喂高热量食物，并从小进行极其严苛的力量训练


  # 六、社会分层差异

  长辈/氏族议员:
    权威来源: 掌握的极品矿脉、失传的锻造图纸和家族规模
    性观念: 极其包容且豪迈 / "只要不耽误明天打铁，晚上怎么折腾都行"

  矿工与初级铁匠:
    联山国绝对主体·日复一日在地下与岩石打交道
    性观念: 粗犷、直接、充满力量感 / 喜欢在刚开采完的矿洞里或堆满煤炭的角落进行，认为沾满煤灰和汗水的结合最受炉母祝福

  蒸汽工程师与高级符文师:
    掌控联山国核心科技的群体
    性观念: 充满硬核实验精神 / 喜欢在亲密行为中引入各种自制的蒸汽辅助机械，追求极致的物理刺激与效率

  武器商人与雇佣兵（客场）:
    在人类城市或纳维里斯倾销武器的群体
    性观念: 劳累一天后，极度渴望伴侣的粗暴抚慰和烈酒 / 常因动作过大砸坏人类旅馆的家具而赔钱

  人类与精灵的影响:
    人类的复杂恋爱观在矮人看来纯属"吃饱了撑的"。精灵的优雅在矮人看来是"毫无用处的花架子"。矮人依然我行我素地保持着大口喝酒、大声咆哮、用力打铁的硬核作风
