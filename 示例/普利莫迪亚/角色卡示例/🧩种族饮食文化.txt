种族饮食文化: # 各种族生理和文化导致的饮食差异。只记录AI无法从其他设定推导的生理特征和感官差异。

  人类三国: # 基线杂食，无特殊感官优势或盲区。调味基础盐/芥末/香草
    饮酒: # 酒精耐受中等。贵族讲究葡萄酒配正餐，平民日常麦酒
  矮人: # 味觉阈值偏高，同样咸度矮人觉得刚好人类觉得偏咸。酒精耐受所有种族最高
    饮酒: # 酒精耐受所有种族最高。外族第一次喝矮人酒往往被辣到或醉倒
  精灵: # 味觉极精细，能分辨同种香料不同产地。寿命长一个味道可唤起数百年记忆
    饮酒: # 饮酒极慢，一杯品一整夜。对酒的纯净度要求近乎洁癖
  暗精灵: # 菌菇特化，能辨别上百种菌菇细微差异。暗视强亮处视觉受限，判断食物更依赖嗅觉味觉
    饮酒: # 菌菇酒是日常饮品也是社交工具。不讲究饮酒礼仪
  龙裔: # 体温高，人类觉得烫嘴龙裔觉得刚好温热。辛辣耐受最高
    饮酒: # 体温高偏好热饮酒。饮酒有等级礼仪，不同血脉浓度喝不同档次
  珑族: # 体小代谢高食量相对不小。灶火崇拜——冷食在文化中不吉利
    饮酒: # 酒精耐受低几杯就脸红。冷酒文化不吉利必须热饮
  半人马: # 进食量极大。边走边吃偏好能单手拿的食物形态
    饮酒: # 边跑边喝。酒壶是随身装备。不存在"品酒"只有"喝酒"
  拉弥亚: # 分叉舌感知空气化学分子，嗅味灵敏度极高。进食频率极低可数周不食但单次量极大。对食材新鲜度感知远超其他种族
    饮酒: # 单次饮量可观。蜜酒分享有严格等级含义——大母蛇祭司先饮
  那伽: # 舌头感知气味能力强，对咸鲜平衡敏感度极高。潮汐节律影响食欲
    饮酒: # 潮汐节律影响饮酒——涨潮时饮量增大

  兽族:
    犬族: # 嗅觉是人类数十倍，香气层次感知远超其他种族。嗅觉过载会不适。味觉不如嗅觉精细
      饮酒: # 杯没端到嘴边已闻完全部香气层次
    狐族: # 嗅觉仅次犬族但偏分析型——拆解各层成分而非整体冲击。对复合调味鉴赏力兽族中最高
      饮酒: # 最擅长品酒的兽族。能精确拆解酒香各层并给出毒舌点评
    猫族: # 甜味迟钝（缺受体）。对鲜味脂肪香极敏感
      饮酒: # 蜜酒"就是水"。对任何酒都不如热奶有兴趣
    薮猫族: # 甜味迟钝同猫族。暗视能力最强，夜间看食物色彩比其他种族更丰富。昼伏夜出
      饮酒: # 更在意酒在暗光下的色泽表现
    狸猫族: # 杂食适应性最强。对水分含量极敏感——菜的水分是否恰好能精确判断
      饮酒: # 什么酒都能接受。掺水的酒一口就知道
    兔族: # 苦味极敏感，微量苦味即引起不适。听觉极敏，厨房声音先影响对菜的期待
      饮酒: # 啤酒花苦的麦酒是挑战。偏好甜型果酒和蜜酒。酒精耐受低
    鹿族: # 天生草药香料鉴赏家，能指出用了什么香草甚至产自什么季节。鹿角在进食时会微微发热
      饮酒: # 药草酒/香料酒鉴赏家——能指出浸泡了什么草药及浸泡时间。好酒时鹿角微热
    羊族: # 对气压敏感延伸到味觉，能品出发酵程度细微差异。发酵食品鉴赏力所有种族最高
      饮酒: # 发酵酒鉴赏力最高。对酿造缺陷(醋酸/硫化物)感知最敏锐
    牛族: # 力量耐力强，食量对应最大。对饱腹感满足阈值最清晰
      饮酒: # 饮量大。清淡果酒评价"跟水一样"
    狼族: # 嗅觉与犬族同源但更粗犷，擅长远距离捕捉整体气味。群体进食有本能秩序——首领先食
      饮酒: # 群体饮酒首领先端杯。远距离就能闻到酒馆开了什么酒
    虎族: # 蛋白质鲜味敏感度最高，肉的火候差五秒能吃出来。对肉的部位和质地有本能级判断力
      饮酒: # 偏好高酒精度浓烈饮品。清淡果酒"不过瘾"
    狮族: # 鲜味敏感度同虎族。进食时有强烈领地意识。对食物的"气势"有要求——精致小份会觉得被看轻
      饮酒: # 对酒的"气势"有要求——大碗大壶才对。不满意时吼声让整个酒馆都知道
    鸟族: # 味蕾少味觉不灵敏，更依赖视觉判断食物。能看到人类看不到的色彩细节。对质地/口感比味道更在意
      饮酒: # 更在意酒的外观——色泽、透亮度、气泡表现
    鳄族: # 口腔内部极敏感，对口感质地分辨力所有种族最高。变温——寒冷时热食是生理需要。进食频率低单次量极大
      饮酒: # 对酒的口感质地(酒体轻重/单宁粗细/气泡大小)分辨力最高。寒冷时热酒是生理需要
    蜥蜴族: # 变温生物，寒冷时热食是生理续命。舌头感知气味（嗅味合一），进食前伸舌"尝"空气中的食物信息
      饮酒: # 进饮前伸舌"尝"空气中酒香。寒冷时热酒是生理续命。偏好矿物质底味突出的酒
    熊猫族: # 兽族中攻击性最低。对竹笋类/植物纤维有独特偏爱。甜味感知比其他兽族稍强
      饮酒: # 蜜酒和甜型果酒接受度高。不挑酒。喝慢

  极北三族:
    约顿: # 极寒狩猎传统，偏好大型猎物和动物脂肪
      饮酒: # 蜂蜜酒分量以桶计
    雪女族: # 极地渔猎传统
      饮酒: # 冰封酒极珍贵偶为外交礼物
    佛摩尔族: # 峡湾海洋采集传统
      饮酒: # 峡湾海藻酒外族几乎无法入口

  其他种族:
    人鱼: # 纯海洋饮食体系
      饮酒: # 深海发酵饮品对陆地种族口味极陌生
