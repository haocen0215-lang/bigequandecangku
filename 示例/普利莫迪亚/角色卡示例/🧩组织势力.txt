组织势力: # 大陆主要组织势力概览

  行业公会: # 各行业从业者的职业组织，制定行业规范，保护成员利益

    蒸汽工会: # 成立仅五年，增长最快的新兴势力。总部克朗港，人类三国和矮人山国活跃。收拢近八成蒸汽从业者，入会门槛低培训周期短，组织风格年轻激进。

    锻造公会: # 历史最悠久的行业公会之一，与矮人氏族议会关系深厚。严格的学徒-行者-大师制度，培养合格铁匠需十年以上。掌握附魔铭刻和特种金属加工等不可替代的高端工艺。

    商业公会: # 最有钱的行业组织，控制主要商路贸易秩序和集市管理。在韦斯托利亚势力最强，实质上控制港口共和制的议会多数席位。在卡尔德里亚和阿尔登马克受封建贵族制约。

    运输商会: # 陆上运输从业者联盟，掌控马车队、驿站网络和货物保险。与商业公会既合作又争主导权——商会控制贸易秩序，运输商会控制物流能力。

    矿工公会: # 传统矿工和深界浅层资源采集者的组织。矮人矿工是骨干，人类矿工数量更多。与锻造公会是原材料供应链上的天然盟友。与冒险者工会有明确管辖划分：矿工管采集，冒险者管探索战斗。

    酿造师公会: # 酒类和发酵食品行会。规模不大但渗透力强，几乎每个有酒馆的城镇都有存在。掌控酿造标准、原料采购渠道和品牌认证。

    药剂师公会: # 炼金和药剂制作者行会。掌握药剂配方、炼金器具标准和成品品质认证。因深界稀有草药供应持续增加，地位近年显著上升。与冒险者工会有稳定的材料-加工合作关系。

  跨国中立机构: # 不隶属于任何种族政权、跨大陆运作的中立组织

    冒险者工会: # 总部纳维里斯，负责深界探索、任务发布、冒险者等级认定。中立铁律禁止参与国家军事行动。

    魔法师协会: # 跨种族魔法学术和职业组织。总部纳维里斯，各种族领地设分会。职能包括学术交流、执业资质认证、危险魔法监管。精灵领地分会实质独立运作。

  学术机构: # 各种族高等教育体系

    诸族学院: # 各种族领地内高等教育机构总称。同时是年轻精英的社交场，贵族、富商、新兴家族子女在此建立人脉网络。详细学院体系将在专题图谱中展开。

  灰色势力: # 游离于合法秩序之外的地下经济和犯罪网络

    黑市网络: # 不是统一组织，而是走私商、腐败官员、部分冒险者和更深处势力在地下交汇形成的网络。纳维里斯最密集。表层是经济犯罪（走私、伪造认证、违规采集物），深层远比表面肮脏。血族与黑市有千丝万缕的联系。
