职业体系: # 力量光谱（纯物理←→纯魔法）上的战斗职业与光谱外的制造职业
  战斗职业: # 从纯物理到纯魔法的光谱分布，每个职业占据区间而非点，个体可在区间内偏移
    战士: # 光谱最左（物理95/魔法5）；技术+经验+装备+意志的极致；含专精方向: 狂战士（觉醒狂暴）、苦修者（觉醒苦修之力）、重甲骑士、武器专精等；种族体质差异在此职业体现最明显
    游荡者: # 偏左（物理90/魔法10）；敏捷技巧隐蔽为核心——潜行/暗杀/机关/开锁/城市行动；高阶者可能学少量辅助小魔法但非核心；盗贼和刺客是社会角色或专精方向非独立职业
    圣骑士: # 偏左（物理75/魔法25）；重甲近战+信仰之力——物理系与信仰型的交叉点；通过信仰型渠道获有限魔法（自我强化/驱邪/少量治疗）；是信仰之力的最典型载体，所有物理系中最容易觉醒特殊能量
    游侠: # 偏左（物理70/魔法30）；野外追踪+弓术/陷阱+少量自然魔法——物理系与自然型的交叉点；猎人是平民到见习级野外生存者，接受系统训练后向游侠过渡
    牧师: # 偏右（物理35/魔法65）；行走的信仰实践者——可穿中甲甚至重甲前线支援；擅长战斗中即时治疗/强化/驱邪；冒险队最常见的治疗支援角色
    德鲁伊: # 偏右（物理30/魔法70）；自然感应+变形+环境操控——通过自然型渠道深度接入；自然环境中极强但人造环境中减弱；变形能力让高阶德鲁伊近战输出也很强
    祭司: # 靠右（物理15/魔法85）；驻守型仪式施法者——大型祝福/净化/结界，需准备时间但效果更持久强大；不适合一线但军队祝福/城镇结界/重大祭仪不可替代
    巫师: # 靠右（物理10/魔法90）；契约型施法者——力量来自与超凡存在互利协议，契约直接赋能但范围受限且需履行义务；契约对象决定能力特色
    术士: # 靠右（物理10/魔法90）；血脉型施法者——天生通道以本能情感驱动；血脉领域天花板极高但选择面窄；无法自由选择发展方向
    法师: # 光谱最右（物理5/魔法95）；学术型施法者——纯知识驱动；知识面最广但需大量研究积累；身体素质明显短板，高度依赖团队保护；法术书是核心装备
  制造职业: # 不在力量光谱上但产出直接决定战斗者装备上限和恢复效率
    工匠: # 传统制造业者（锻造/木工/皮革等），不使用或仅极微量魔力；装备产业链基础层
    魔工匠: # 锻造+附魔结合，通过工艺型渠道接入魔力场；核心能力是将魔法效果固化到物质载体——附魔武器/符文铠甲/防护符文等；与蒸汽工匠是不同职业但可融合
    蒸汽工匠: # 蒸汽机械设计制造和维护，通过蒸汽型渠道接入魔力场；蒸汽时代新兴职业；可与魔工匠技能融合
    炼金术士: # 物质转化+药剂制作，通过工艺型渠道接入魔力场；产出含恢复药剂/强化药剂/战斗消耗品/深界原料提纯；代价系统中"恢复枯竭"的核心供给者
