食味体验: # 食物感官体验的维度索引。包含正面和负面体验。AI据词条自行理解含义并描写。

  基础味觉:

    咸:
      正面: # 矿物粗咸、海水清咸、发酵深咸、烟熏回甘咸、盐花细咸
      负面: # 齁咸、咸苦、咸涩、咸腥、死咸（咸但没有鲜味托底的空洞咸）
      程度: # 寡淡→提味→口重→齁咸→咸苦发涩

    甜:
      正面: # 蜂蜜花甜、焦糖苦甜、果实清甜、麦芽温厚甜、奶甜、根茎自然甜
      负面: # 齁甜、腻甜、甜腥、糊甜（焦糖烧过头的苦甜）、空甜（只甜没有层次）
      程度: # 若有若无→明确愉悦→发腻→齁甜→甜到反胃

    酸:
      正面: # 柑橘明亮酸、醋的穿刺酸、发酵圆润酸、浆果温和酸、酒酸
      负面: # 酸涩、酸腐、酸倒牙、尖酸（过于尖锐刺激的酸）、馊酸
      程度: # 提亮清新→生津开胃→面部紧缩→酸倒牙→胃酸上涌

    苦:
      正面: # 焦化烟苦、草药清苦、啤酒花苦、坚果暗苦、茶苦回甘
      负面: # 焦苦、涩苦、药苦、生苦（未熟蔬菜/生豆）、碳化苦
      程度: # 增添深度→入口苦随即回甘→明确苦涩→本能想吐→嘴里残留苦味

    鲜:
      正面: # 肉汤厚重鲜、菌菇深沉鲜、海产清透鲜、发酵复合鲜、骨髓浓鲜
      负面: # 腥鲜、膻鲜、假鲜（调味堆砌的浮表鲜）、鲜过头（嘴发黏不适）、腐鲜（不新鲜食材的鲜味和腐味交织）
      程度: # 说不上来但好吃→每一口都满足→鲜味过密→腥味浮现→鲜到不适

    辣:
      正面: # 辣椒灼热、胡椒穿刺、姜的温暖辣、芥末冲鼻辣、花椒麻辣
      负面: # 辣过头（纯灼烧）、辣不均（一口极辣一口没味）、空辣（只辣没有底味）、辣到失味
      程度: # 尾端温热→嘴唇发麻→满头大汗→口腔丧失味觉→只剩灼痛

    涩:
      正面: # 单宁收敛涩、茶的清涩、红酒果涩
      负面: # 麻涩、干涩、生柿涩、过度收敛
      程度: # 干净收束→切油腻→嘴巴塞棉花→无法吞咽

    麻:
      正面: # 花椒酥麻、轻麻回暖
      负面: # 麻木、麻到失去知觉、嘴唇肿胀感
      程度: # 嘴唇轻酥→持续震颤→说话受影响→口腔麻木

  复合味觉:
    酸甜: # 正面: 互相救赎 | 失败: 酸甜分离、甜盖酸、酸盖甜
    咸鲜: # 正面: 最基础的好吃 | 失败: 咸盖鲜、鲜过头发黏、只咸不鲜
    苦甘: # 正面: 先苦后甜 | 失败: 只苦不甘、苦味持续不退
    麻辣: # 正面: 震颤加灼烧 | 失败: 麻盖辣、辣盖麻、麻辣空洞无底味
    酸辣: # 正面: 又湿又热 | 失败: 酸辣腥、酸辣打架
    甜咸: # 正面: 最上瘾组合 | 失败: 甜咸打架每口摇摆、甜咸腻
    鲜辣: # 正面: 鲜铺底辣点睛 | 失败: 辣杀鲜、鲜辣腥
    酸鲜: # 正面: 酸劈开鲜的沉闷 | 失败: 酸腐鲜、馊味

  口感质地:

    正面脆系: # 酥脆、焦脆、薄脆、爆裂脆
    正面柔系: # 绵软、蓬松、丝滑、入口即化
    正面弹韧系: # Q弹、糯韧、筋道、有嚼劲
    正面对比系: # 外酥里嫩、外焦里糯、先脆后软、表焦内润
    正面挂壁包裹系： # 挂壁、包裹、贴合、轻薄挂杯、均匀覆盖
    正面乳化丝滑系： # 丝滑、丝绒、匀净、油水交融、无感油润
    正面凝胶弹力系： # 凝胶、Q弹、回弹、胶质感、果冻状
    正面浓缩厚重系： # 浓稠、挂齿、粘稠、厚实、风味缓释、久留不散
    正面空气泡沫系： # 蓬松、轻盈、泡沫感、入口即化、云朵般质地
    正面镜面光泽系： # 镜面、反光、晶莹、亮泽、油亮诱人
    正面清润化渣系： # 清润、化渣、净口、迅速消散、不留黏腻感
    

    负面脆系: # 受潮发皮、硬壳、焦壳空心、油腻酥（炸物吸油过多的假酥）
    负面软系: # 水烂、糊状、黏糊、稀烂（汤菜的固体部分完全失去形态）
    负面韧系: # 嚼不动、橡胶感、柴散、干硬、皮筋感
    负面对比系: # 外焦内生、外软内硬、出水塌陷、外干内冰（没解冻就烹饪）
    负面水油分离系： # 分层、油腻、割裂、寡淡水味、浮油
    负面粉质颗粒系： # 粉糊、生粉味、砂感、颗粒感、粗糙磨损
    负面失控粘滞系： # 胶水感、黏滞、坨块、无法流动、吞咽困难
    负面凝结残次系： # 过稠、固态、结块、疙瘩、厚重油膜、胶着感
    负面稀薄寡淡系： # 稀薄、清汤、挂不住、风味流失、水感过重
  体验时序: # AI按此顺序选择性展开
    视觉期待: # 正面: 色泽诱人/热气升腾/油光发亮 | 负面: 颜色灰暗/焦黑/惨白/塌陷/汤底浑浊/摆盘散架
    嗅觉前奏: # 正面: 远香→近香→揭盖爆发 | 负面: 远处就闻到焦糊/腥气/酸腐/甜腥/杂乱
    入口第一味: # 正面: 触感先行→前味爆发→惊喜 | 负面: 第一口就"不对"——冲突/空洞/过度
    咀嚼展开: # 正面: 口感变化→味觉深化→混合 | 负面: 越嚼越暴露问题——腥味释放/嚼不动/藏在里面的生或焦
    吞咽余韵: # 正面: 暖意/回味/肩膀放松 | 负面: 不想吞/嘴里残留怪味/胃部不适/想喝水冲刷
