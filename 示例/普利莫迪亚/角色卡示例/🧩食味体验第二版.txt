食味体验: # 食物感官体验的维度索引。包含正面和负面体验。AI据词条自行理解含义并描写。

  味觉核心: # 舌面与口腔最直接感知到的味道/刺激。不等同于严格生理学基础味觉，而是写作用核心口味维度。

    咸:
      正面: # 矿物粗咸、海水清咸、发酵深咸、烟熏回甘咸、盐花细咸、汤底托味咸
      负面: # 齁咸、咸苦、咸涩、咸腥、死咸（咸但没有鲜味托底的空洞咸）、盐粒刺舌
      程度: # 寡淡→提味→口重→齁咸→咸苦发涩

    甜:
      正面: # 蜂蜜花甜、焦糖苦甜、果实清甜、麦芽温厚甜、奶甜、根茎自然甜、酒酿甜
      负面: # 齁甜、腻甜、甜腥、糊甜（焦糖烧过头的苦甜）、空甜（只甜没有层次）、甜到发黏
      程度: # 若有若无→明确愉悦→发腻→齁甜→甜到反胃

    酸:
      正面: # 柑橘明亮酸、醋的穿刺酸、发酵圆润酸、浆果温和酸、酒酸、乳酸清爽酸
      负面: # 酸涩、酸腐、酸倒牙、尖酸（过于尖锐刺激的酸）、馊酸、胃酸感
      程度: # 提亮清新→生津开胃→面部紧缩→酸倒牙→胃酸上涌

    苦:
      正面: # 焦化烟苦、草药清苦、啤酒花苦、坚果暗苦、茶苦回甘、黑巧苦甜
      负面: # 焦苦、涩苦、药苦、生苦（未熟蔬菜/生豆）、碳化苦、苦味残留不散
      程度: # 增添深度→入口苦随即回甘→明确苦涩→本能想吐→嘴里残留苦味

    鲜:
      正面: # 肉汤厚重鲜、菌菇深沉鲜、海产清透鲜、发酵复合鲜、骨髓浓鲜、酱香鲜
      负面: # 腥鲜、膻鲜、假鲜（调味堆砌的浮表鲜）、鲜过头（嘴发黏不适）、腐鲜（不新鲜食材的鲜味和腐味交织）
      程度: # 说不上来但好吃→每一口都满足→鲜味过密→腥味浮现→鲜到不适

    甘:
      正面: # 茶后回甘、谷物温甘、草药清甘、肉汤尾甘、焦糖余甘、酒后回甘
      负面: # 假回甘、甜味剂残甘、甘腻、苦后不甘、喉咙黏甜
      程度: # 隐约回甜→喉间生津→余味悠长→甘腻黏喉

    辣:
      正面: # 辣椒灼热、胡椒穿刺、姜的温暖辣、芥末冲鼻辣、花椒麻辣、油辣厚辣
      负面: # 辣过头（纯灼烧）、辣不均（一口极辣一口没味）、空辣（只辣没有底味）、辣到失味、灼痛
      程度: # 尾端温热→嘴唇发麻→满头大汗→口腔丧失味觉→只剩灼痛

    麻:
      正面: # 花椒酥麻、轻麻回暖、舌尖震颤、麻中带香
      负面: # 麻木、麻到失去知觉、嘴唇肿胀感、麻涩、麻味发苦
      程度: # 嘴唇轻酥→持续震颤→说话受影响→口腔麻木

    涩:
      正面: # 单宁收敛涩、茶的清涩、红酒果涩、清爽收束、切油腻
      负面: # 麻涩、干涩、生柿涩、过度收敛、嘴巴塞棉花
      程度: # 干净收束→切油腻→嘴巴塞棉花→无法吞咽

    辛:
      正面: # 葱姜蒜辛香、胡椒辛香、芥末冲鼻辛、萝卜清辛、香料辛暖
      负面: # 生蒜冲鼻、辛辣刺喉、呛咳、辛味压香、鼻腔刺痛
      程度: # 微微提神→鼻腔打开→喉头发热→呛咳→刺激压倒味觉

    凉:
      正面: # 薄荷清凉、瓜果水凉、冷汤爽凉、草本凉感、冰镇清爽
      负面: # 凉得发苦、薄荷牙膏味、冷腥、冷油感、香气打不开
      程度: # 舌尖微凉→口腔清爽→鼻腔通透→凉到刺舌→冷感破坏风味


  风味气息: # 嗅觉、鼻后香气、食材来源感、发酵感、动物性气息、烹饪气息。

    香:
      正面: # 炙烤肉香、麦香、奶香、果香、草本香、坚果香、香料暖香、油脂香
      负面: # 香精味、香过头、闷香、廉价甜香、香气刺鼻、香气杂乱
      程度: # 淡淡飘出→入口扩散→满口留香→香气压味→闻久发腻

    腥:
      正面: # 海鱼清腥、贝类海潮腥、血肉铁腥、鱼露发酵腥、蟹黄脂腥
      负面: # 死鱼腥、血腥刺鼻、腐腥、腥臭、腥到反胃、冷腥
      程度: # 海味隐现→明确动物性→压过鲜味→腐败感浮现→本能抗拒

    膻:
      正面: # 羊肉温膻、奶酪醇膻、野味厚膻、兽脂香膻、牧场气息
      负面: # 羊骚膻、陈脂膻、皮毛膻、膻臭、膻到发苦、膻味压香料
      程度: # 动物脂香→风味明显→压住调味→骚膻上头→难以下咽

    臭:
      正面: # 发酵醇臭、奶酪霉臭、豆腐臭香、鱼露咸臭、榴莲甜臭、糟香臭
      负面: # 腐臭、酸臭、馊臭、霉臭、垃圾臭、下水道臭
      程度: # 若隐若现→风味独特→冲鼻→压倒食欲→恶心反射

    烟熏:
      正面: # 木柴烟香、果木熏香、培根烟咸、炭火余烟、草烟野性、茶熏清烟
      负面: # 烟灰味、呛烟、焦油味、熏过头、苦烟味、灰尘感
      程度: # 轻烟尾韵→烟香包裹→明显熏制→刺喉→像吃灰

    酒:
      正面: # 果酒甜香、红酒酸涩、烈酒辛暖、米酒圆润、麦酒苦香、酒糟香
      负面: # 酒精刺鼻、酒苦、酒酸败、酒味未挥发、醉腻、酒味割裂
      程度: # 隐约酒香→暖意浮现→酒味支撑菜肴→酒精感突出→刺鼻烧喉

    青:
      正面: # 青豆清香、青草气、黄瓜青甜、嫩叶清青、草药青苦、新鲜植物气息
      负面: # 生青、草腥、青涩、未熟果青、豆腥青、像嚼草
      程度: # 清新→植物感明确→青涩→生味压舌→像嚼草

    土:
      正面: # 蘑菇湿土香、根茎泥土甜、甜菜矿物土味、野菜土青、雨后泥土气
      负面: # 泥腥、土臭、霉土味、没洗干净的沙土感、尘土味
      程度: # 自然朴实→根茎感明显→野性土味→泥腥→像吃土

    陈:
      正面: # 陈醋厚酸、老酒陈香、陈年奶酪、风干肉熟成味、豆酱陈香、老卤香
      负面: # 霉味、旧油味、灰尘味、腐败陈味、仓库味、过期味
      程度: # 轻微熟成→陈香明显→复杂深沉→霉味浮现→腐败感

    铁:
      正面: # 血肉铁鲜、肝脏矿物感、铁锅锅气、野味血香、矿物咸鲜
      负面: # 血腥铁锈味、金属苦、铁腥、硬币味、舌面发涩
      程度: # 隐约矿物感→血肉感增强→铁味明显→金属苦涩→像舔铁器

    焦:
      正面: # 焦糖香、锅气焦香、炭烤焦香、烤面包焦香、肉边焦脆
      负面: # 烧焦、碳化苦、糊底味、焦黑刺喉、烟灰味
      程度: # 微焦提香→焦香明确→烟熏感→焦苦出现→碳灰味残留

    脂:
      正面: # 牛油厚脂、猪油圆润脂、奶油柔脂、坚果油香、鱼脂清润、骨髓脂香
      负面: # 油腻、哈喇味、腻喉、浮油感、冷油糊嘴、脂肪腥
      程度: # 润口→丰腴→满足感强→发腻→胃里沉重

    清:
      正面: # 清甜、清鲜、清酸、清香、汤底清透、无杂味
      负面: # 寡淡、空清、水味、没有支撑、清到无聊
      程度: # 干净→通透→轻盈→单薄→寡水

    厚:
      正面: # 汤底厚、酱汁浓、肉味扎实、奶油厚重、发酵层次深、骨汤厚
      负面: # 闷厚、糊嘴、沉重、味道堵住、浓到发腻
      程度: # 有底味→味道扎实→层次浓密→压舌→吃几口就累


  结构型风味: # 味道之间的搭配关系。正面是互相支撑，失败是互相打架。

    酸甜: # 正面: 互相救赎、清亮开胃、甜托酸柔 | 失败: 酸甜分离、甜盖酸、酸盖甜、像糖水兑醋
    咸鲜: # 正面: 最基础的好吃、底味扎实、鲜味被咸味托出 | 失败: 咸盖鲜、鲜过头发黏、只咸不鲜
    苦甘: # 正面: 先苦后甜、层次成熟、余味回转 | 失败: 只苦不甘、苦味持续不退、药味压口
    麻辣: # 正面: 震颤加灼烧、热烈上瘾、香气有底 | 失败: 麻盖辣、辣盖麻、麻辣空洞无底味
    酸辣: # 正面: 又湿又热、刺激开胃、酸劈开辣的厚重 | 失败: 酸辣腥、酸辣打架、尖酸加灼痛
    甜咸: # 正面: 最上瘾组合、甜托咸香、咸压甜腻 | 失败: 甜咸打架、每口摇摆、甜咸腻
    鲜辣: # 正面: 鲜铺底辣点睛、辣味激活鲜味 | 失败: 辣杀鲜、鲜辣腥、鲜味被辣味烧空
    酸鲜: # 正面: 酸劈开鲜的沉闷、鲜味更清楚 | 失败: 酸腐鲜、馊味、酸把鲜味打散

    咸脂: # 正面: 咸味托住油脂、越吃越香、脂香有骨架 | 失败: 咸油腻、冷脂咸、油盐分离
    酸脂: # 正面: 酸解油腻、清爽复食、脂肪变轻 | 失败: 酸油分离、尖酸浮油、油腻仍压舌
    苦脂: # 正面: 黑巧、咖啡奶油、烤坚果感、成熟厚味 | 失败: 苦油、旧油苦、苦腻
    鲜脂: # 正面: 骨髓、鱼腹、肉汤油花、鲜味被脂肪延长 | 失败: 腥脂、黏腻鲜、鲜到糊嘴
    焦甜: # 正面: 焦糖、烤南瓜、烤洋葱、暖甜加深 | 失败: 糊甜、焦苦压甜、烧糖味
    焦苦: # 正面: 咖啡、炭烤边缘、烟火深度 | 失败: 碳化、烧糊、灰苦、焦黑刺喉
    烟咸: # 正面: 熏肉、火腿、烟熏盐、烟香拉长咸味 | 失败: 像吃烟灰盐、苦烟咸、咸烟刺喉
    酒酸: # 正面: 红酒炖肉、果酒酱汁、酸香圆润 | 失败: 酒酸败、醋酒打架、酒精尖酸
    酒甜: # 正面: 蜜酒、甜酒甜品、酒香托甜 | 失败: 酒精甜腻、甜酒刺鼻、醉甜
    青苦: # 正面: 草药、嫩叶、橄榄油、植物清苦 | 失败: 生苦、草腥、青味压口
    青涩: # 正面: 茶、未熟果轻涩、清爽收束 | 失败: 生柿子涩、嘴干、青涩刮舌
    腥鲜: # 正面: 海鲜、贝类、鱼露、海潮鲜 | 失败: 死鱼腥、腐鲜、腥味盖住鲜味
    膻鲜: # 正面: 羊汤、野味、兽脂鲜 | 失败: 骚膻压鲜、膻臭发苦
    臭鲜: # 正面: 臭豆腐、发酵鱼露、蓝纹奶酪、腐乳鲜 | 失败: 腐臭鲜、酸臭鲜、臭味压倒食欲
    辛甜: # 正面: 姜糖、肉桂甜点、香料暖甜 | 失败: 甜辛刺喉、香料压甜、甜味发冲
    辛鲜: # 正面: 葱姜蒜爆香、芥末海鲜、辛香提鲜 | 失败: 辛味杀鲜、生蒜压鲜、呛鼻
    凉甜: # 正面: 薄荷糖、冰镇瓜果、冷甜清爽 | 失败: 牙膏甜、凉感压甜、冷甜发腻
    铁鲜: # 正面: 血肠、肝脏、野味肉汁、矿物鲜 | 失败: 血腥金属味、铁锈鲜
    土甜: # 正面: 烤根茎、甜菜、胡萝卜、泥土甜香 | 失败: 泥甜、土腥、像没洗干净
    陈酸: # 正面: 陈醋、老酒、发酵果酸、酸味变厚 | 失败: 霉酸、馊酸、陈腐酸
    香脂: # 正面: 油脂携带香气、香味贴舌延长 | 失败: 香油腻、脂肪闷香
    清鲜: # 正面: 汤清味鲜、轻盈不寡淡 | 失败: 清到无味、鲜味浮在表面
    厚鲜: # 正面: 骨汤、浓汤、炖煮复合鲜 | 失败: 鲜厚发黏、浓到压舌
    辣脂: # 正面: 辣油、红油、脂肪托辣香 | 失败: 辣油糊嘴、油辣空洞
    酸涩: # 正面: 果皮、茶、红酒的收束感 | 失败: 酸涩刮舌、倒牙又干口


  口感质地: # 咬合、流动、包裹、融化、干湿、弹性、残留。

    正面:

      脆系: # 酥脆、焦脆、薄脆、爆裂脆、清脆、脆壳、轻裂
      柔系: # 绵软、蓬松、丝滑、入口即化、软嫩、柔润、云朵感
      弹韧系: # Q弹、糯韧、筋道、有嚼劲、回弹、紧实、弹牙
      对比系: # 外酥里嫩、外焦里糯、先脆后软、表焦内润、壳脆芯软
      挂壁包裹系: # 挂壁、包裹、贴合、轻薄挂杯、均匀覆盖、酱汁贴舌
      乳化丝滑系: # 丝滑、丝绒、匀净、油水交融、无感油润、圆润
      凝胶弹力系: # 凝胶、Q弹、回弹、胶质感、果冻状、轻轻颤动
      浓缩厚重系: # 浓稠、挂齿、粘稠、厚实、风味缓释、久留不散
      空气泡沫系: # 蓬松、轻盈、泡沫感、入口即化、云朵般质地、轻盈扩散
      镜面光泽系: # 镜面、反光、晶莹、亮泽、油亮诱人、糖壳透亮
      清润化渣系: # 清润、化渣、净口、迅速消散、不留黏腻感、入口即散
      脂润系: # 油润、丰腴、脂香包裹、入口融化、唇齿留香、润而不腻
      汁水系: # 爆汁、多汁、汤汁充盈、咬开流汁、肉汁外溢、内里湿润
      粉糯系: # 糯软、粉甜、细腻、沙糯、根茎绵密、米粉吸汁
      纤维松散系: # 酥烂、肉丝散开、纤维顺滑、轻轻一抿就散
      薄膜爆裂系: # 薄膜包裹、轻咬爆开、汁液弹出、内外反差
      冰爽系: # 冰凉、清爽、紧实、冰晶轻裂、冷而不腻

    负面:

      脆系: # 受潮发皮、硬壳、焦壳空心、油腻酥、回软、脆而刮口
      软系: # 水烂、糊状、黏糊、稀烂、软塌、塌陷
      韧系: # 嚼不动、橡胶感、柴散、干硬、皮筋感、越嚼越累
      对比系: # 外焦内生、外软内硬、出水塌陷、外干内冰、壳肉分离
      水油分离系: # 分层、油腻、割裂、寡淡水味、浮油、油水两张皮
      粉质颗粒系: # 粉糊、生粉味、砂感、颗粒感、粗糙磨损、粉粒刮舌
      失控粘滞系: # 胶水感、黏滞、坨块、无法流动、吞咽困难、粘牙
      凝结残次系: # 过稠、固态、结块、疙瘩、厚重油膜、胶着感

  AI使用规则:
    - 描写食物时优先选择1到3个味觉核心，不要堆满。
    - 风味气息用于说明食材身份、烹饪方式、发酵程度、动物性气息或异常气味。
    - 结构型风味用于判断味道是否协调。正面写互相支撑，失败写互相打架。
    - 口感质地用于说明咬合、流动、包裹、融化、干湿、弹性与残留。
    - 工艺食味用于连接烹饪手法和最终体验，不需要为每一种烹饪手法单独创造新体验。
    - 体验时序用于组织描写顺序，不必每次完整展开。
    - 正面体验可以混入少量危险边缘感，例如轻腥、轻膻、轻苦、轻涩、轻烟，以增加真实感。
    - 负面体验不一定代表难吃，也可以代表角色不习惯、食材低劣、火候失败、保存不当或文化差异。
    - 同一道菜最多选择1个主要工艺食味，辅以1到2个次要工艺食味，避免体验堆叠混乱。
    - 若烹饪手法复杂，优先判断最终入口结果，而不是机械列出所有手法。
    - 热菜优先强化嗅觉前奏、入口第一味、咀嚼展开；冷菜优先强化视觉期待、入口第一味、吞咽余韵。
    - 炸、煎、烤类优先检查表面结构；炖、焖、煮、蒸类优先检查水分状态；腌、发酵、熟成类优先检查渗透和余味。
    - 描写失败食物时，不要只写“难吃”，应指出失败来源：过火、欠火、失水、吸油、发酵过度、腌渍失衡、香气封闭、油水分离等。
    - 食材属性、烹饪手法、调味结构、工艺食味、口感质地、体验时序应共同决定最终描写。
