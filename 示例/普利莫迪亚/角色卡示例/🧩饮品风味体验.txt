饮品风味体验: # 饮品特有的感官维度。与食味体验互补——基础味觉（酸甜苦鲜涩麻）和体验时序直接引用食味体验，此处只收录饮品独有的感官通道。

  外观:
    正面: # 清澈透亮、色泽丰富饱满、细密持久气泡、优雅挂杯痕迹、光泽油润
    负面: # 浑浊不清、颜色灰暗氧化、沉淀物漂浮、气泡粗大短命、色泽发褐

  酒精感:
    正面: # 柔和温暖→温热舒适→热辣刺激（随酒精度递增）
    负面: # 酒精刺鼻辣喉（酒体太薄撑不住酒精度）、灼烧感过强

  酒体:
    轻盈如水→中等丰满→浓厚饱满→黏稠厚重

  碳酸感:
    静态→微泡→活泼→强气泡→爆裂

  单宁:
    正面: # 丝滑收敛、柔和干净、结构感强
    负面: # 粗糙干涩、绿色涩感（未成熟原料）

  余韵:
    长度: # 极短（入喉即消）→短→中→长→极长（分钟级回味）
    变化: # 单调→展开（新风味浮现）→逆转（先甜后苦/先果后辛）

  制作缺陷:
    发酵缺陷: # 醋酸变质、硫化物臭、杂醇超标溶剂感、发酵不充分水味
    工艺缺陷: # 生青味（原料未熟）、煮烂味（温度过高）、过度氧化纸板味、浑浊沉淀
    冲泡缺陷: # 过萃苦涩、欠萃寡淡、水温不当（烫熟茶叶/萃取不足）
